# Sentimental-Analysis
Sentiment analysis is a Natural Language Processing (NLP) technique used to determine the emotional tone of text. Classifies text as Positive, Negative, or Neutral using predefined word lists. Interactive input allows real-time analysis.

Features:

Interactive user input for real-time sentiment analysis.<br>
Pure Python implementation, no external libraries required.<br>
Lightweight and Easy to learn.<br>

Applications:

Customer feedback analysis<br>
Social media comment analysis<br>
Basic chatbot sentiment detection<br>

Conclusion:<br>
This project provides a foundation for understanding how AI can interpret human emotions from text. It can be extended with machine learning or deep learning models for more advanced sentiment analysis.
